Alan Mo - ART 2060 Final

Digital Erosion

Hover over an image to animate.

Artist Statement

This project explores the intersection of code and visual art through the manipulation of images using sorting algorithms. Digital Erosion uses JavaScript to create interactive images that respond to user input. These show how an image can be transformed from it's original state into something new and possibly unexpected.

When the user hovers over each canvas, each column of the image is sorted based on the luminance of each pixel in that column. Depending on the original image, this can create a variety of effects, from smoothing and blurring to more chaotic and fragmented appearances.

Some images were specifically selected to have very straight and geometric lines, so that you are able to see the result of sorting the columns based on luminance. Other images were selected to have more organic and complex patterns which results in something more unexpected when the columns are sorted.
